0. File Maintenance and Version Control

We'll begin by getting to know our way around GitHub and becoming familiar with the basics of Python. Since students are coming to the course with different levels of experience with Python, we will structure things a little differently. If you are already fairly proficient in Python, you can spend this time becoming familiar with GitHub, or helping your peers who are new to Python. Items 1 and 2 should be prioritised. Items 3-5 do not need to be completed within module 0, but it's recommended you attempt them at some point in the course once you're comfortable with Python.

  1. Explore GitHub and complete Asst-0
  2. Review Python Topics that will be on Quiz 0
  3. Create a GitHub page (like this one) by creating a gh-pages branch from your own Repository. You can use your github page to organise your assignments or for whatever you like. GitHub hosts your pages for free!
  4. If you want to get fancy with your READMEs, and generally be more stylish, look at Markdown Syntax, used to style .md files
  5. See me for a challenge (i.e. help me troubleshoot GitHub ahaha)

Optional Python Practice Exercise: Write a program that asks the user to input 10 integers, and then prints the largest odd number that was entered. If no odd number was entered, it should print a message to that effect

1. Python Loops, Lists, and Strings

The main focus of this Module will be problem solving using Python Loops, Lists, and String Manipulation. Since it is relevant to string manipulation, we will also briefly review number systems, character mapping, and byte sequences. Following is your to-do/to-know list for this module

  1. Get your GitHub page up and running (See Module 0)
  2. Be familiar with Python Topics included on Quiz 1: Lists, Loops, and Strings
  3. Here's a pretty good Python resource for these topics
  4. Keep usng GitHub workflow. If you're still confused, see freecodecamp's videos below
  5. Be familiar with various Number Systems (Unary/Binary/Hexadecimal/Decimal), their advantages and their uses
  6. Understand how character mapping works, in general terms, with ASCII, UNICODE, UTF-8, UTF-16
  7. Complete as many of the Python Practice Exercises as you can. They are intended to help you develop problem-solving and coding skills. THerefore, it would be a waste of your time to look up or copy others' solutions without attempting the problems yourself
  8. Here's some more review of Python Lists, whether you're learning it for the first time, or just need a refresher, it's good practice for the quiz
